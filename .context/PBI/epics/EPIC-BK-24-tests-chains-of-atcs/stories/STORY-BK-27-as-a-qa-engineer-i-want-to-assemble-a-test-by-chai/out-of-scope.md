## Out of scope

- Reordering the ATC chain after the Test is created (covered by a separate story)
- Loading the Test's full step + assertion detail in a single view (covered by a separate story)
- Editing a Test's title or chain after creation (backlog)
- Deleting or archiving a Test (backlog)
- Duplicating an existing Test as a starting point for a new one (backlog)
- Running the Test (lives in the Manual Runs epic, BK-006)
- Bulk-creating multiple Tests in one operation (out of MVP entirely)
- Parametrized Tests with data tables or equivalence partitions (Phase 3)
- Test version history beyond the activity log (Phase 2)
- Live notification to teammates the moment a Test is created (covered when the Realtime epic lands)
